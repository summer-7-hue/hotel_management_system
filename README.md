Hotel Management System (Database Project)

Overview
This project is a Hotel Management System (HMS) developed as part of the Database Systems course.  
It provides a centralized database solution to manage hotel operations like reservations, guests, staff, billing, and inventory.

 Objectives
- Automate hotel operations
- Reduce data redundancy
- Maintain data integrity using normalization (up to 3NF)
- Provide efficient querying and reporting

 Key Features:
 
 Guest Management
- Store guest details, contacts, and ID proofs
- Maintain guest history

 Reservation System
- Booking, check-in, check-out
- Room availability tracking

 Room Management
- Room types, pricing, and capacity
- Status tracking (Available / Occupied)

 Staff Management
- Employee records and departments
- Role-based structure

 Billing & Payments
- Invoice generation
- Payment tracking

 Inventory Management
- Track stock, suppliers, purchases

 Maintenance & Housekeeping
- Room cleaning schedules
- Maintenance requests

 Database Design

- ERD & EERD included
- Fully normalized up to 3NF
- Uses:
  - Primary Keys
  - Foreign Keys
  - Constraints


 Technologies Used

- Oracle SQL
- PL/SQL
- ER Modeling
- Normalization Techniques
